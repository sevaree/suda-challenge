# 溯答挑战

一款记忆力训练游戏，复刻自 Nintendo 3DS《脑锻炼5分钟魔鬼特训》中的经典玩法。

## 游戏规则

**核心玩法：记住答案，回答过去**

1. **记忆阶段**：屏幕上显示一道算术题（如 `3 + 5 = ?`），你有 2 秒时间记忆答案
2. **答题阶段**：等溯答 N 题后，系统会出一道新题，但要求你回答**第 N 道题的答案**

### 示例（溯答 3 题）

| 阶段 | 显示内容 | 操作 |
|------|---------|------|
| 第1题（记忆） | `1: 3 + 5 = ?` | 记住答案 8 |
| 第2题（记忆） | `2: 7 - 2 = ?` | 记住答案 5 |
| 第3题（记忆） | `3: 9 - 6 = ?` | 记住答案 3 |
| 第4题（答题） | `4: 2 + 4 = ?` | 回答第1题：**8** |
| 第5题（答题） | `5: 6 + 1 = ?` | 回答第2题：**5** |
| ... | ... | ... |

## 游戏设置

- **溯答题数**：1-99 可选，数值越大难度越高
- **游戏时间**：2 分钟
- **题目范围**：10 以内加减法（答案 0-9）

## 计分规则

- 答对：基础 10 分 + 难度加成（溯答数 × 2）
- 答错：可继续尝试，直到正确为止
- 正确率 ≥ 90% 获得最高评价
